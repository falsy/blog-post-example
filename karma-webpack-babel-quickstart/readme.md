# Karma Webpack Babel quick start

Karma 테스트 러너로 ES6+ 의 코드를 유닛테스트 하기. 추가로 Webpack, Babel, Jasmine을 사용합니다.

## 참고 문서

https://falsy.me/karma-테스트-러너로-es6-의-코드를-유닛-테스트하기/

## 설치 및 실행

### 설치

```
$ npm install
```

### 카르마 테스트 러너 실행

```
$ npm test
```

### 빌드

$ npm run build
